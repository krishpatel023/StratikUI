# Scroll To Top
### This primitive allows you to smoothly scroll to the top of the page. It can be used to navigate to the top of the page when a certain threshold is reached. You can implement this feature by including the `<ScrollToTop />` component in your application. This will automatically detect if the user has scrolled past the specified threshold and will scroll to the top of the page enabling clean code implementation.
