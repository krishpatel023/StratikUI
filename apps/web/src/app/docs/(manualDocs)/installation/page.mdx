# Installation

<PARTITION className="mt-8">
## Requirements
</PARTITION>

- [React 18](https://react.dev/learn/installation)  or higher
- [Next.js 14](https://nextjs.org/docs/getting-started) or higher
- [TailwindCSS 3.4](https://tailwindcss.com/docs/installation) or higher
- [Framer Motion 11](https://www.framer.com/motion/introduction/) or higher
- [React Aria Components 1.3](https://react-spectrum.adobe.com/react-aria/getting-started.html) or higher
